# Movie Database Data Cleaning and Analysis Project

This project focuses on importing, cleaning, and analyzing a comprehensive movie dataset to uncover insights about profitability, viewer preferences, and top-rated global cinema. The codebase is implemented entirely in Python using the pandas and numpy libraries.

## Project Overview
The assignment involves working through programmatic steps to process raw unstructured movie data into clean datasets. Key analysis goals include discovering the top 10 most profitable movies, identifying top-performing foreign language films, and curating an IMDb Top 250 subset based on explicit user voting thresholds.

## Data Cleaning Pipeline
To prepare the dataset for authentic insights, a rigorous cleaning process was executed:
* **Inspecting Missing Values:** Evaluated column-wise and row-wise null value percentages.
* **Dropping Redundant Columns:** Stripped away descriptive or unneeded social media tracking attributes like color, director_facebook_likes, and actor_1_facebook_likes, leaving 13 analytics-ready variables.
* **Null-Row Treatment:** Eliminated rows with critical gaps in high-null percentage areas—specifically filtering out records missing essential financial information like gross and budget.
* **Data Imputation:** Imputed structural missing text values, such as filling missing data inside the language column safely with "English".
* **Retention Validation:** Verified that after executing all cleaning metrics, approximately 77.16% of the original structural rows were perfectly retained.

## Key Insights and Analysis

### 1. Currency Unit Conversions
The numerical scales of financial vectors like gross and budget were normalized from exact absolute dollar counts into standardized Million $ increments to improve data readability.

### 2. Highest Profit Generation
A new metric variable named profit (gross minus budget) was explicitly synthesized. Sorting the dataset highlighted the top most profitable films of all time:
* **Avatar** leads with a profit of 523.51 Million $.
* **Jurassic World** closely follows with 502.18 Million $.
* **Titanic** rounds out the top three with 458.67 Million $.

### 3. IMDb Top 250 Selection
An IMDb_Top_250 dataframe was generated by grouping high-performing titles that safely met a specific audience criteria threshold: an imdb_score rank ordering restricted to movies containing more than 25,000 voted users. 
* **The Shawshank Redemption** ranks at the pinnacle with an IMDb score of 9.3.
* **The Godfather** holds the runner-up slot with a score of 9.2.

### 4. Top Foreign Language Cinema Excellence
By subsetting non-English films out of the elite top tier list, standout foreign cinematic projects were isolated:
* **The Good, the Bad and the Ugly** (Italian, Score: 8.9)
* **City of God** (Portuguese, Score: 8.7)
* **Seven Samurai** (Japanese, Score: 8.7)
* **Spirited Away** (Japanese, Score: 8.6)
